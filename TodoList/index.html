The Todo List API App is an ASP.NET Web API application that's been published as an Azure API App. This is a sample HTML client that communicates with a REST API hosted in Azure App Service.

Azure API Apps make use of Swagger metadata to describe REST API endpoints. The Swagger UI link above will load an easy-to-use API test page. The Swagger JSON link above will load the Swagger metadata.

The Generate C# Client link above will use AutoRest to generate a C# client for this API App. To use this C# in your own Visual Studio project, install the AutoRest Client Runtime NuGet into your project and then import the generated files, and you can call this API App directly from almost any C# project.
